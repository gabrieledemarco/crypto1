# Crypto1 Project

## Project Architecture (Graphify)

This repository is indexed by **Graphify**. The knowledge graph is generated in the `graphify-out/` directory and includes:

- `graph.json` – the full graph in JSON format
- `GRAPH_REPORT.md` – a concise markdown report summarizing the main nodes, communities, and knowledge gaps
- `graph.html` – an interactive 3‑D visualisation you can open in a browser.

**How to view the graph**
```powershell
start "" "$(pwd)\graphify-out\graph.html"
```
On macOS/Linux use `open $(pwd)/graphify-out/graph.html`.

The graph is automatically refreshed on every commit via a Git hook (see step 5). It serves as the permanent mental map for anyone working on this codebase.
